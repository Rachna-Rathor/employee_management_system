# Employee Management System (EMS)

A fully functional **Employee Management System** built using **React.js**, **Tailwind CSS**, and **Context API**. This project demonstrates practical front-end development, role-based dashboards, authentication, and task management with data persistence using `localStorage`.

---

## Features

### 1. Role-Based Dashboards
- Separate dashboards for **Admin** and **Employees**.
- Dynamic UI rendering based on user role.

### 2. Authentication Flow
- Login functionality for both employees and admins.
- Maintains **currently logged-in user state**.
- Logout feature with session handling.

### 3. Task Management
- Create, assign, and update tasks.
- Task statuses: **New**, **Active**, **Completed**, **Failed**.
- Task filtering and dynamic updates across components.

### 4. State Management
- Centralized **state using React Context API**.
- Avoids prop drilling and enables efficient data flow.

### 5. Data Persistence
- Uses **localStorage** to store tasks and user data.
- Maintains state across page reloads.

### 6. UI/UX
- Responsive and clean UI using **Tailwind CSS**.
- Modular, reusable, and scalable component structure.
